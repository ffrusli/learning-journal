# Learning Journal

This repository documents all my cybersecurity and IT learning activities.  
It includes logs, notes, hands-on exercises, and useful resources.

## Structure
- `logs/` – Daily or weekly learning logs
- `notes/` – Topic summaries and study notes
- `projects/` – Small practical exercises and labs
- `resources/` – Useful links, references, and tools

## Purpose
- Track my progress
- Keep all learning in one place
- Build a habit of documentation
- Create a record of continuous improvement
